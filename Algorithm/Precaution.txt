알고리즘 문제 풀기 전 유의사항

1. 입출력은 scanf, printf로 받자 -> cin,cout 속도 느림
// 굳이 써야겠다면 endl 대신 "\n" 사용하기 -> 10,000,000 이정도로 많은 입력에서는 쓰면 시간초과남

2. 자료구조 기초 알고리즘들은 라이브러리를 이용하자 ex) 벡터, 리스트, 스택, 큐, 트리, 정렬 등등

3. 다양한 기준을 가지고 정렬을 하려면 클래스를 이용하여 객체를 생성후 bool Operator 를 이용하여 비교해준다

4. 크기가 너무 클 경우 적당한 자료형을 쓰자 ex) 졸라 큰 정수형 int -> long long 사용

5. 재귀함수를 돌릴때는 메모리제이션을 이용하여 이미 값을 알고 있는 재귀함수는 다시 돌리지 않는 것을 이용하여 풀어야 한다.

6. 크기가 너무 커서 나머지값을 구하라고 하는 경우에는 계산을 할 때마다 계속 %로 나머지를 구해줘야 백준에 정답 넣을 때 안터진다

7. 크기가 너무 큰 배열을 만들어야할 때는 벡터로 만들어서 차지하는 공간을 줄이면 만들 수 있다.

8. 입력이 없는 데이터의 경우 cin.eof()함수를 이용하여 판별한다.
